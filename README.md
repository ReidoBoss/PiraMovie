# 🎬 PiraMovie

**PiraMovie** is a personal movie streaming website I built for my girlfriend and friends — and now I'm sharing it with the world!  
It’s powered by **Nuxt 3** (frontend + backend with Nitro) and integrates with the **TMDB API** for movie data.

## 🚀 Tech Stack

- **Frontend:** [Nuxt 3](https://nuxt.com/) + Tailwind CSS  
- **Backend:** [Nuxt Nitro](https://nitro.unjs.io/)  
- **Movie Data:** [TMDB API](https://developer.themoviedb.org/)  
- **Player:** HTML5 video player  

---

## 🌟 Highlights

- 🎨 **Modern UI** – Clean and minimal design.  
- 🎥 **TMDB Integration** – Fetches movie info, posters, and details dynamically.  
- ⚡ **Full Nuxt Stack** – One codebase handles both frontend and backend.  
- 🔒 **Private Streaming** – Designed for a small circle of friends and family.
